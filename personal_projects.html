Game Archive

Web Development · Interactive Games

In Progress

Game Archive is a retro-inspired web application that allows users to browse and play a growing collection of simple browser-based games and activities.

The project was designed to recreate the nostalgic feeling of classic game collections while keeping the interface approachable and easy to navigate. Current activities include rock-paper-scissors, hangman, coin flipping, and dice rolling.

Beehavior-inator

Cybersecurity · Artificial Intelligence

Functional Prototype

Beehavior-inator is a functional honeypot system built to capture and analyze potentially malicious user activity.

The software records attacker interactions and sends structured log information to a large language model. The model then organizes the logs, explains notable behavior, and provides suggestions for improving how the honeypot is configured and monitored.

This project allowed me to explore the relationship between cybersecurity monitoring, automated log interpretation, and AI-assisted analysis.

  • Cybersecurity
  • Honeypot
  • Python
  • LLM Integration
  • Log Analysis

Ghost House

Unity · Game Development

Completed Project

Ghost House is a complete Unity game made up of three connected gameplay sections that flow together as one experience.

  • An anomaly-detection section that challenges players to identify unusual environmental changes
  • A seven-room puzzle-solving sequence
  • A parkour-focused platforming section

The project required level planning, game logic, interaction design, testing, and coordination between multiple gameplay systems.

  • C#
  • Level Design
